## Web Get Bot
Simple bot to fetch web page. Just to learn implementing Telegram Bot.

## Get Started

```
virtualenv wgetbot
cd wgetbot
git clone https://github.com/k4ml/wgetbot.git
cd wgetbot
../bin/python main.py <BOT TOKEN>
```
